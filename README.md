### Lattice Grid
A high-performance, headless data grid engine built with React and TypeScript, designed for large-scale data rendering and fine-grained control over behavior, state, and UI composition.

Architecture:
- Designed a fully headless core separating state management, business logic, and rendering layers
- Built a modular plugin-based system enabling independent feature composition (sorting, grouping, selection, etc.)
- Implemented controlled and uncontrolled state patterns for maximum flexibility in integration

Core Features:
- Column system with dynamic grouping, resizing, reordering, and visibility control
- Row and column virtualization for efficient rendering of large datasets
- Advanced selection model supporting multi-select and state synchronization
- Fine-grained event system for intercepting and extending grid behavior

Performance Engineering:
- Optimized rendering using memoization, stable references, and minimal re-renders
- Efficient virtualization strategy to handle high row/column counts with low memory footprint
- Batched updates and state isolation to prevent unnecessary UI recalculations

Customization & Extensibility:
- Fully composable API allowing consumers to control rendering at every level
- Pluggable architecture enabling selective feature usage without overhead
- Theme-agnostic design supporting complete UI customization

Impact:
- Enables building highly customized data-intensive interfaces
- Scales efficiently with large datasets and complex interactions
- Serves as a reusable core for multiple data-driven applications
